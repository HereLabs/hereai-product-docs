Mira connects out to the systems your team already runs. The integrations below ship as native, first-class destinations or providers.

## CRM destinations

| Integration | What it delivers |
| --- | --- |
| **[HubSpot](/mira/integrations/hubspot/)** | Contact create / update with custom product properties, plus bidirectional sync |
| **[Salesforce](/mira/integrations/salesforce/)** | Lead or Contact with custom fields, plus bidirectional sync |
| **[Generic webhook](/mira/integrations/generic-webhook/)** | Signed JSON POST to any endpoint you operate, with retry |

## Outbound

| Integration | What it does |
| --- | --- |
| **[ESPs](/mira/integrations/esps/)** | Mailbox / ESP integration for outbound sequences (Outreach, Salesloft, Apollo, custom) |

## Identity & provisioning

| Integration | What it does |
| --- | --- |
| **[SSO](/mira/integrations/sso/)** | Single sign-on via Google or Microsoft (OIDC) |
| **[SCIM](/mira/integrations/scim/)** | Automated user / group provisioning from your IdP |

## Catalog

| Integration | What it does |
| --- | --- |
| **[PIM (Salsify, Productsup)](/mira/integrations/pim/)** | Sync your catalog from a PIM or marketplace tool |
